Icon Placeholder Files

These are temporary placeholder icons for development.

To generate proper icons for production:
1. Create a 1024x1024 PNG icon source file
2. Run: cargo tauri icon path/to/icon-1024x1024.png
3. This will generate all required icon sizes automatically

Required icons:
- 32x32.png
- 128x128.png  
- 128x128@2x.png (256x256)
- icon.icns (macOS bundle)
- icon.ico (Windows)

For now, the app will use default Tauri icons during development.
